# ft_transcendence
 a full-stack webapplication as final project of the Codam core curriculum - 42 school network

A .env file, with at least the following, is needed to run the website:

```
ADMIN=admin
PASSWORD=password

FRONTEND_PORT=5173
BACKEND_PORT=3000
LOG_LEVEL=info

PUBLIC_KEY=123456
PRIVATE_KEY=234567
```
